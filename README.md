# frontend-template
Template for (no framework) frontend projects 

- [DEMO LINK](https://yashnyi.github.io/frontend-template/)

## Description
* Eslint
* Stylelint
* Husky
* Parcel
* Concurrently
* SaSS

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm start`
